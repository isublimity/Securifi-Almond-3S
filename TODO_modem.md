## 1. Оптимизация сигнала

### Блокировка бендов
Текущий модем скачет между бендами. Можно заблокировать на лучший:

```bash
# Проверить текущий
AT+QNWINFO
# Только Band 3 + Band 7 (лучшие для Билайн Москва)
AT+QCFG="band",0,44,0
# Только Band 20 (800 МГц, лучше проникает в помещения)
AT+QCFG="band",0,80000,0
# Все бенды (сброс)
AT+QCFG="band",0,800d5,0
```

## 7. AT-команды — справка

### Диагностика
```
ATI                          — модель/версия
AT+CSQ                       — сигнал (0-31)
AT+QENG="servingcell"        — текущая сота (RSRP, RSRQ, SINR, Band, CellID)
AT+QNWINFO                   — текущий бенд и технология
AT+COPS?                     — оператор
AT+CPIN?                     — статус SIM
AT+QCCID                     — ICCID SIM-карты
AT+CGPADDR=1                 — IP адрес PDN
AT+QGDCNT?                   — счётчик данных
```

### Управление
```
AT+QCFG="band",0,<lte>,0    — блокировка LTE бендов
AT+CFUN=1,1                  — перезагрузка модема
AT+COPS=0                    — автовыбор оператора
AT+CGDCONT=1,"IP","apn"      — настройка APN
```

### Бенды LTE (маски)
| Band | Частота | Маска | Билайн МСК |
|------|---------|-------|-----------|
| B1 | 2100 | 0x1 | Да |
| B3 | 1800 | 0x4 | Да, основной |
| B5 | 850 | 0x10 | — |
| B7 | 2600 | 0x40 | Да |
| B8 | 900 | 0x80 | Да |
| B20 | 800 | 0x80000 | Да, дальнобойный |

### Порты USB
| Порт | Назначение |
|------|-----------|
| /dev/ttyUSB0 | DM (диагностика) |
| /dev/ttyUSB1 | NMEA (GPS, не используется) |
| /dev/ttyUSB2 | AT-команды |
| /dev/ttyUSB3 | PPP (не используется с QMI) |
| /dev/cdc-wdm0 | QMI (основной data-канал) |
